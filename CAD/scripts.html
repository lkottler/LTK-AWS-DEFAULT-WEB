ColorZeroElev.lsp

Stage: EXPERIMENTAL

Command: COLORZEROELEV

This program prompts for entity selection. It CURRENTLY deletes and duplicates all entities with 0 elevation into red copies. This is an experimental version and lacks testing.

CondensePointNums.lsp

Stage: 80% TESTED

Command: CONDENSE_POINTS

This program works by removing all gaps in point numbers in selected points. It uses the smallest point number as the base.

LayerMaker.lsp

Stage: MISSING DCL DIALOGUE BOX

Command: EXCEL_MAKE

Program creates layers from excel .xls with Name, Linetype, Color.
Version 2021-12-01 currently does not prompt for organization of excel columns, assumes E,F,G for Name,Color,Linetype

Command: EXCEL_CONVERT

Converts layers in A,B,C to layers in E,F,G. Iteratively runs MERGE_LAYER command.
format as Name,Color,Linetype

LayerLegend.lsp

Stage: COMPLETE

Command: LAYERLEGEND

Creates a visual for layer status on any drawing.

CurbStake.lsp

Stage: 60% IN PROGRESS

Command: CURBSTAKE

Will stake points of a curb intelligently

Command: STAKE_SINGLEPOINT

Prompts for surface selection, prompts for coordinate, creates cogo point with elevation from coordinate, but point is offset with x,y input.

FilePoke.lsp

Stage: DEV TOOLS

WARNING: This script is mostly for inquiry in how the program is operating, and is useful for creation of more scripts. This is useful only if you know what you are doing.

Command: LAYER_PRINT

Prints dxf of every layer to console.

Command: ENTITY_DUMP

Prompts for entity selection, prints properties of entity to console

Command: ENTITY_PRINT

Similar to ENTITY_DUMP, but simply prints the entity with (print entx), rather than calling (vlax-dump-object entity T) such in ENTITY_DUMP

Command: GET330NAME

(COMPLEX) Prompts for entity selection, prints linked-list name of entity in dxf 330

Command: FEATURE_LINE_DUMP

Experimental printing of feature line DXFs.


Command: FEATURE_LINE_PIS

Prints all PIs in a feature line.

Command: COGO_LIST_COOR

Prints coordinates of all cogo points in selection.
